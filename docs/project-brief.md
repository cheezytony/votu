This is an application that allows users to create and manage voting polls, find and vote on desired polls>
All voters on each poll are easily accessible to the public but you have to be logged in to create/manage polls or vote.

Users can have multiple emails and phone numbers registered. But one has to be set as the active one. Each one has to be verified.

Polls must have pollOptions specified at the point of creation or update.
Polls cannot be voted on until they are marked as active by the creator.
Active polls cannot be modified.
Polls can have an optional expiry date after which the poll will be automatically closed.
Closed polls cannot be voted on or reopened.

No data can or should be deleted. No soft deletes either.
